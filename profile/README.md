# 辰龙操作系统（ChenLongOS）
辰龙操作系统的长期目标是基于清华大学的 [Starry-OS](https://github.com/Starry-OS/) 开发一个发行版，旨在为智能汽车、机器人、物联网等领域的复杂系统，提供高安全、高性能、自主可控的国产操作系统解决方案。

## 快速上手
辰龙操作系统当前是基于[Arceos](https://github.com/arceos-org/arceos)的。如需了解系统概览，可参考 [辰龙操作系统简介](https://www.chenlongos.cn/article/article.html?id=1)；开发入门请查阅文档中的 [快速上手指南](https://chenlongos.com/raspi4-with-arceos-doc/)。
